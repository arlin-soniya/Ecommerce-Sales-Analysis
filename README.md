# E-Commerce Sales Analysis 

## Project Summary
This project analyzes E-Commerce sales data using SQL (Oracle APEX) for backend data processing and Power BI for interactive visualization.

The objective of this project is to extract meaningful business insights, identify sales trends, evaluate product performance, and measure key performance indicators (KPIs) to support data-driven decision-making.

---

## Tools & Technologies
- SQL (Oracle APEX) – Data cleaning, transformation, aggregation
- Power BI – Data modeling, DAX calculations, interactive dashboard development

---

## Project Workflow

### 1. Data Preparation (SQL – Oracle APEX)
- Explored raw dataset structure
- Verified data types and checked duplicates
- Standardized date formats
- Created a structured dataset table
- Added calculated columns:
  - Profit (20% of Total Sales)
  - Cost (80% of Total Sales)

### 2. KPI & Business Analysis (SQL)
Performed aggregation using:
- `SUM()`
- `COUNT()`
- `GROUP BY`
- `EXTRACT()`

Key metrics calculated:
- Total Revenue
- Total Profit
- Total Orders
- Total Quantity Sold
- Profit Margin %

Business analysis performed:
- Monthly sales trend
- Category-wise performance
- Location-wise revenue
- Top 5 products by revenue
- Order status distribution

### 3. Dashboard Development (Power BI)
- Created DAX measures for KPIs
- Designed interactive KPI cards
- Built monthly trend analysis
- Added category & regional comparison charts
- Implemented map visualization
- Added slicers for dynamic filtering
